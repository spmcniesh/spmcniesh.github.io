Full-Stack · Database Design

Workout Tracker

Python PostgreSQL Neon Render

A full-stack web application developed as part of CISC 450 (Database Design) at St. Thomas. Users can log exercises, track sets and reps, and view their workout history through a clean web interface. The backend is built in Python with a PostgreSQL database hosted on Neon, and the application is deployed and served via Render.

Designing the relational schema from scratch — working through entity relationships and building the application around that data model — was the core challenge and learning outcome of this project.

Data Science · Statistical Analysis

NBA Aging Analysis

R ggplot2 Statistics

A data science report exploring how NBA player performance changes with age, using 2023 season statistics. The analysis identifies peak performance years, the rate and shape of post-peak decline, and how aging patterns differ across positions.

Built using R and ggplot2, with statistical methods including regression modeling and grouped aggregation to surface clear trends in a dataset spanning hundreds of player-seasons.

Machine Learning · Research

DASC 400 — Data Mining Final

JMP Minitab Regression Decision Trees k-NN

Final research project for DASC 400 (Data Mining and Machine Learning) at the University of St. Thomas, Spring 2026. Applied multiple data mining techniques — multiple linear regression, decision trees, k-means clustering, and k-nearest neighbors — to a real dataset to answer a defined research question.

Delivered as both a written research paper and a class presentation. The project required not just running the models, but interpreting results rigorously and communicating findings clearly to a non-specialist audience.
