# Personal Portfolio Website

## Overview
A personal portfolio website built with HTML and CSS, hosted via GitHub Pages at
[spmcniesh.github.io](https://spmcniesh.github.io). The site includes a home page,
a projects showcase, a résumé, and an about page — built from scratch without any
frontend framework as a demonstration of core web development fundamentals.

## Installation Instructions
No installation required — this is a static site hosted on GitHub Pages. To run locally:
1. Clone the repository:
```bash
   git clone https://github.com/spmcniesh/spmcniesh.github.io.git
```
2. Open `index.html` in any browser. No build step or server needed.

## Usage Instructions
The site has four pages navigable via the top menu:
- **Home (`index.html`)** — Introduction, skills overview, and project previews
- **Projects (`projects.html`)** — Full descriptions of coursework and personal projects
- **Résumé (`resume.html`)** — Academic background, work experience, and skills
- **About (`fun.html`)** — Personal interests and hobbies outside of school

Live site: [https://spmcniesh.github.io](https://spmcniesh.github.io)
